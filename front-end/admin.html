Документация использования

Руководство по работе с админ-панелью

🚀

👥
Управление группами

Добавление группы:

  • Нажмите кнопку "Добавить группу"
  • Введите название группы (например: "ПИ 25-1.2")
  • Добавьте студентов с их данными
  • Сохраните группу

Редактирование группы:

  • Разверните нужную группу
  • Нажмите кнопку "Редактировать"
  • Внесите необходимые изменения
  • Сохраните изменения
💡 Совет
Используйте кнопку "Развернуть все" для одновременного просмотра всех групп
🎓
Управление студентами

Добавление студентов:

  • При создании/редактировании группы добавляйте студентов
  • Указывайте ФИО студента (обязательно)
  • Добавляйте номер телефона (опционально)
  • Отмечайте статус оплаты

Изменение статуса оплаты:

  • Разверните группу для просмотра студентов
  • Статус "Оплатил" отображается зеленым цветом
  • Статус "Не оплатил" отображается красным цветом
  • Для изменения статуса отредактируйте группу
📊
Статистика и отчеты

В разделе "Статистика" доступна следующая информация:

  • Общее количество групп
  • Общее количество студентов
  • Количество оплативших студентов
  • Количество студентов с задолженностью
  • Процент оплаты в реальном времени
📈 Аналитика
Статистика обновляется автоматически при каждом изменении данных
💡
Советы и рекомендации
  • Регулярно сохраняйте изменения
  • Используйте понятные названия групп
  • Ведите актуальные контактные данные студентов
  • Своевременно обновляйте статусы оплаты
  • Для безопасности регулярно меняйте пароль доступа
