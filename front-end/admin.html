Управление группами

Добавление группы:

  • Нажмите кнопку "Добавить группу"
  • Введите название группы (например: "ПИ 25-1.2")
  • Добавьте студентов с их данными
  • Сохраните группу

Редактирование группы:

  • Разверните нужную группу
  • Нажмите кнопку "Редактировать"
  • Внесите необходимые изменения
  • Сохраните изменения
💡 Совет
Используйте кнопку "Развернуть все" для одновременного просмотра всех групп
🎓
Управление студентами

Обязательные поля для студентов:

  • ФИО студента - обязательно для заполнения
  • Статус оплаты - автоматически определяется при импорте

Дополнительные поля:

  • Номер телефона - опционально, для связи
  • Форма оплаты - указывается при импорте из Excel

Изменение статуса оплаты:

  • Разверните группу для просмотра студентов
  • Статус "Оплатил" отображается зеленым цветом
  • Статус "Не оплатил" отображается красным цветом
  • Для изменения статуса отредактируйте группу
📁
Импорт и экспорт данных

Импорт из Excel:

  • Нажмите кнопку "Excel Импорт/Экспорт" в разделе групп
  • Выберите файл Excel для загрузки
  • Система автоматически определит группы и студентов
  • Обязательные колонки в Excel: "Группа" и "ФИО студента"
  • Дополнительные колонки: "Форма оплаты", "Моб. Телефон", "Телефон"

Требования к Excel файлу:

  • Формат: .xlsx или .xls
  • Максимальный размер: 10MB
  • Первая строка должна содержать заголовки колонок
  • Обязательные колонки: Группа, ФИО студента
  • Колонка "Форма оплаты" определяет статус оплаты автоматически

Экспорт в Excel:

  • Нажмите "Excel Импорт/Экспорт" → "Экспорт в Excel"
  • Скачайте полный отчет по всем группам
  • Файл содержит: №, Группа, ФИО студента, Статус оплаты, Телефон, Дата добавления
📋 Пример структуры Excel
Обязательные колонки: Группа | ФИО студента | Форма оплаты | Моб. Телефон
⚡
Автоматическое определение оплаты

При импорте из Excel:

  • Бесплатники - автоматически получают статус "Оплачено"
  • Платники - автоматически получают статус "Не оплачено"
  • Система анализирует колонку "Форма оплаты"

Ключевые слова для определения:

  • Оплачено: "бесплатно", "free"
  • Не оплачено: "платно", "оплачено", и другие значения
🔄 Логика работы
Все платники при импорте получают статус "Не оплачено" для последующего ручного контроля оплат

Статистика и отчеты

В разделе "Статистика" доступна следующая информация:

  • Общее количество групп
  • Общее количество студентов
  • Количество оплативших студентов
  • Количество студентов с задолженностью
  • Процент оплаты в реальном времени
📈 Аналитика
Статистика обновляется автоматически при каждом изменении данных
🔒
Безопасность и обработка данных

Защита данных:

  • Доступ к административной панели защищён паролем и реализован с помощью сессионной авторизации (Express Session)
  • Все данные передаются по защищенному соединению
  • Автоматическая фильтрация конфиденциальных данных (ИИН)

Обработка персональных данных:

  • Система автоматически игнорирует колонки с ИИН
  • Хранятся только необходимые данные: ФИО, телефон, статус оплаты
  • Возможность массового управления через Excel
🛡️ Конфиденциальность
Система разработана с учетом требований к защите персональных данных
