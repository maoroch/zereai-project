## Что делает бот

Этот бот отправляет студентам уведомления о необходимости оплатить учёбу и позволяет отправлять чек в формате PDF.
Если студент оплатил, бот обновляет статус в базе.



## 🧰 Что нужно перед запуском

### 1. Установить Node.js (если нет)

Скачать отсюда → [https://nodejs.org](https://nodejs.org)
Проверить:

node -v
npm -v

### 2. Создать файл `.env` в корне проекта

Создать файл и положить туда:

```
BOT_TOKEN=токен_telegram_бота
SUPABASE_URL=https://xxxxx.supabase.co
SUPABASE_ANON_KEY=анон_ключ
SUPABASE_SERVICE_KEY=service_role_ключ
```

Токен Telegram бота берём у @BotFather
Ключи — в Supabase → Project Settings → API

---

### 3. Положить Google JSON ключ в корень проекта

Файл должен называться **`cryptic-episode-476814-g2-88ac169c4d89.json`**

Если у друга файл с другим именем → надо заменить имя в коде:

```js
keyFile: "ИМЯ_ФАЙЛА.json"
```

---

### 4. Установить зависимости

В терминале в папке проекта:

```
npm install
```

---

## ▶️ Как запустить

```bash
node bot.js
```

Если всё ок — в терминале будет:

```
✅ Бот запущен
```

---

## ❗ Важно

Если бот говорит:

```
409: Conflict
```

значит бот **уже запущен в другом окне** → закрыть старый терминал или перезапустить компьютер.

---

## 📂 Структура Storage

PDF файлы сохраняются в Supabase Storage → Bucket `files`

```
files/payments/<id_студента>/payment.pdf
```

---

## 🧪 Как проверить бота

1. Найти бота в Telegram → нажать **Start**.
2. Появятся две кнопки:

   * ✅ Прочитал сообщение
   * 💳 Оплата за учёбу
3. Нажать «Оплата за учёбу».
4. Отправить **PDF** (не фото, не скрин).
5. Бот обработает → статус студента обновится на `paid = true`.

---

## 🆘 Если что-то не работает

| Проблема             | Решение                                    |
| -------------------- | ------------------------------------------ |
| Бот не отвечает      | Проверить BOT_TOKEN                        |
| Не грузится PDF      | Проверить `SUPABASE_SERVICE_KEY`           |
| Вылетает при запуске | Проверить `.env` и имя JSON файла Google   |
| Конфликт 409         | Бот уже запущен → закрыть другие терминалы |
